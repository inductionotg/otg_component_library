Buttons

Use OTG UI custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more

Default Buttons

                    	
                    		<button class="btn btn-primary">Primary</button>
				<button class="btn btn-secondary">Secondary</button>
				<button class="btn btn-success">Success</button>
				<button class="btn btn-error">Error</button>
				<button class="btn btn-disable" disabled>Disabled</button>

Outline Buttons

                    	
                    		<button class="btn btn-outline-primary">Primary</button>
				<button class="btn btn-outline-secondary">Secondary</button>
				<button class="btn btn-outline-success">Success</button>
				<button class="btn btn-outline-error">Error</button>
				<button class="btn btn-outline-disable" disabled>Disabled</button>

Link Buttons

                       
                        <button class="btn"><a href="" class="btn-link btn-link-primary">Primary Link</></button>
                        <button class="btn"><a href="" class="btn-link btn-link-secondary">Secondary Link</a></button>

                                <span class="button__text">Download</span>
