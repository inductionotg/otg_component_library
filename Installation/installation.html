Installation

For using OTG UI in your projects, you just have to paste the below line of code in the head section of your HTML file and you are ready to use all components. 

						
                            <link rel="stylesheet" href=" https://otgui.netlify.app/style.css">

Starter Template

						
                            <!DOCTYPE html>
                            <html lang="en">
                            <head>
                                <meta charset="UTF-8">
                                <meta http-equiv="X-UA-Compatible" content="IE=edge">
                                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                                <title>OTG UI</title>
                                <link rel="stylesheet" href="href=" https://otgui.netlify.app/style.csss">  
                            </head>
                            <body>
                                
                            </body>
                            </html>
